Política de Privacidade

Esta extensão foi desenvolvida com o único propósito de auxiliar na validação de formulários em páginas web.

1. Coleta e uso de informações

- A extensão acessa somente os elementos da página ativa para executar os testes de validação.
- Rótulos de campos (textos curtos) podem ser enviados para a API pública do LanguageTool, exclusivamente para verificação ortográfica.
- Nenhuma outra informação pessoal ou de navegação do usuário é coletada.

2. Armazenamento local

- Os resultados dos testes são salvos apenas no armazenamento local do navegador (chrome.storage.local).
- Esses dados não são enviados a servidores externos.

3. Compartilhamento de dados

- Nenhuma informação coletada pela extensão é compartilhada com terceiros, exceto os textos enviados ao LanguageTool para verificação ortográfica.
- O LanguageTool possui sua própria política de privacidade, disponível em: https://languagetool.org/legal/privacy.

4. Segurança

- A extensão não executa código remoto.
- A extensão não coleta dados de outras abas além da que o usuário seleciona.
